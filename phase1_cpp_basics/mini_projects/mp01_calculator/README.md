# Simple CLI Calculator

## Description
A command-line calculator supporting +, -, *, / on two numbers.

## Topics Applied
Topics 1-2 (Setup, Basics & Syntax)

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp01_calculator main.cpp
./mp01_calculator
```
