# Temperature Converter

## Description
Converts between Celsius, Fahrenheit, and Kelvin using multiple functions; intentionally debugged.

## Topics Applied
Topics 3-4 (Functions & Files, Debugging)

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp02_temperature_converter main.cpp
./mp02_temperature_converter
```
