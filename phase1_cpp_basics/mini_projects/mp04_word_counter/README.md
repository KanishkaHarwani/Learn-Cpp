# Word Counter / Palindrome Checker

## Description
Reads text input and counts words / checks for palindromes using std::string.

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp04_word_counter main.cpp
./mp04_word_counter
```
