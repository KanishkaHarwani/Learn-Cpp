# Pointer Playground

## Description
Manual swap via pointers, a small dynamic array simulation using raw pointers.

## Topics Applied

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp10_pointer_playground main.cpp
./mp10_pointer_playground
```
