# Generic Min/Max/Swap Utility

## Description
Template functions for min, max, and swap that work across types.

## Topics Applied
Topics 11-12 (Type Conversion, Overloading & Templates)

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp09_generic_min_max_swap main.cpp
./mp09_generic_min_max_swap
```
