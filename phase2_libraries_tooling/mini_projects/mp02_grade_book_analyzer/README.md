# Grade Book Analyzer

## Description
Analyzes grades stored in both std::vector and std::array, comparing dynamic vs fixed containers.

## Topics Applied
Topics 3-4 (std::vector, std::array/C-style)

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp02_grade_book_analyzer main.cpp
./mp02_grade_book_analyzer
```
