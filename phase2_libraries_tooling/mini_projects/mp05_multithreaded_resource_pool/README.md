# Multithreaded Resource Pool

## Description
A thread-safe resource pool using smart pointers and std::mutex.

## Topics Applied
Topics 9-10 (Move/Smart Pointers, Multithreading Basics)

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp05_multithreaded_resource_pool main.cpp
./mp05_multithreaded_resource_pool
```
