# Custom Dynamic Array Class

## Description
A hand-built dynamic array class with iterator support and manual memory management.

## Topics Applied

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp03_custom_dynamic_array main.cpp
./mp03_custom_dynamic_array
```
