# Fraction / Vector2D Calculator

## Description
A Fraction and/or Vector2D class with full operator overloading and lambda-based utilities.

## Topics Applied
Topics 7-8 (Advanced Functions, Operator Overloading)

## How to Build & Run
```bash
cd src
g++ -std=c++17 -Wall -Wextra -o mp04_fraction_vector2d_calculator main.cpp
./mp04_fraction_vector2d_calculator
```
